#Data_Science_Masters_Dissertation_UEL

Core objective 
The goal is to build an ML-augmented gravity model of trade for bilateral exports among Kenya, Uganda, and Rwanda (2017–2024 quarterly panel data) to:
1) Predict bilateral trade flows.
2) Quantify the impact of NTBs.
3) Compare traditional gravity estimation against modern machine learning approaches.
4) Maintain interpretability using SHAP.
